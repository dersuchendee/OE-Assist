Below is an example SPARQL query that, under the closed world assumption, retrieves the songs played in a given performance (replace <performance_URI> with the specific performanceâ€™s URI):

PREFIX : <http://www.example.org/test#>
PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#>

SELECT ?song ?songLabel
WHERE {
  ?songPerformance a :Cl_SongPerformance .
  ?songPerformance :performedAt <performance_URI> .
  ?songPerformance :performedSong ?song .
  OPTIONAL { ?song rdfs:label ?songLabel . }
}
